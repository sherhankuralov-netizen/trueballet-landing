# TrueBallet — лендинг франшизы

Одноэкранный конверсионный лендинг для продажи франшизы школы классического балета TrueBallet предпринимателям в регионах Казахстана. Единственная цель — заявка (имя, телефон, город), которая уходит в WhatsApp основателя.

Собран на чистом **HTML + CSS + JS** — без сборки и зависимостей. Деплоится как статический сайт.

## Структура

```
├── index.html    # разметка всех 10 секций + готовый текст из ТЗ
├── styles.css    # дизайн-система и стили (mobile-first)
├── script.js     # отправка формы в WhatsApp, анимации, sticky-CTA
└── README.md
```

## Секции (по ТЗ)

1. Hero — оффер + CTA
2. Боль → Решение
3. Цифры / факты
4. История основателя
5. Философия бренда
6. Что входит во франшизу
7. Достижения и география
8. Срочность + CTA
9. Форма заявки + контакты
10. Footer

## Как запустить локально

Просто открой `index.html` в браузере. Или подними локальный сервер:

```bash
python3 -m http.server 8000
# затем открой http://localhost:8000
```

## Как задеплоить

Любой статический хостинг: **GitHub Pages, Netlify, Vercel, Cloudflare Pages** или обычный shared-хостинг. Просто загрузи три файла в корень.

## Что нужно поменять под себя

### 1. Фото (сейчас — плейсхолдеры-градиенты)
- **Фото основателя** (Блок 4): в `index.html` найди `.founder__photo`. Замени градиентный блок на `<img src="images/erlan.jpg" alt="Ерлан Сыздыков" />` и, при желании, убери подпись-подсказку `.founder__photo-hint` в CSS.
- **Hero и фоны:** для реальных фото (пуанты, дети у станка, светлый зал) добавь `background-image` в правила `.hero`, `.founder__photo` в `styles.css`.

Рекомендованные фото по ТЗ: балетные пуанты, дети у станка, светлый зал, педагог с ребёнком, конкурсные выступления.

### 2. Номер WhatsApp / телефон
Номер `+7 701 777 36 84` указан в нескольких местах:
- `script.js` — переменная `WHATSAPP_PHONE` (формат без `+`, только цифры) — **сюда уходит заявка**;
- `index.html` — ссылки `tel:`, `wa.me`, `t.me` и подписи.

### 3. Как работает форма
При отправке формы открывается WhatsApp основателя с предзаполненным сообщением:
```
Здравствуйте, Ерлан! Хочу узнать условия франшизы TrueBallet.
Имя: … / Телефон: … / Город: …
```
Текст сообщения настраивается в `script.js` (переменная `message`).

## Дизайн-система

| Токен | Значение |
|---|---|
| Чёрный | `#0A0A0A` |
| Жемчужно-белый | `#F5F0EB` |
| Золото | `#C9A84C` |
| Заголовки | Playfair Display (serif) |
| Тело | Inter (sans-serif) |

Все цвета и шрифты вынесены в CSS-переменные в начале `styles.css` (`:root`).

## Особенности

- **Mobile-first** — основной трафик с телефонов.
- **Фиксированная золотая CTA-кнопка** внизу экрана на мобайле (прячется у формы).
- Мягкие анимации появления секций (с уважением к `prefers-reduced-motion`).
- SEO-мета и Open Graph заполнены по ТЗ (Часть 3).
- Цена франшизы не указана нигде — только на звонке (по требованию ТЗ).
